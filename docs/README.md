# Docs Directory Guide

This directory is organized by document purpose.

## Folders

- `api/`
  - frontend-facing and backend-facing API contracts
- `architecture/`
  - cross-cutting source-of-truth documents and architecture references
- `architecture/solana/`
  - Solana-specific architecture and validation plans
- `plans/`
  - implementation plans grouped by domain
- `analysis/`
  - investigation notes, portfolio tracking, and review summaries
- `fixes/`
  - targeted fix plans and follow-up notes
- `runbooks/`
  - manual operator workflows and local test procedures
- `testing/`
  - test scenarios and validation matrices

## Plan Subfolders

- `plans/ai/`
  - AI decisioning and related implementation plans
- `plans/backend/`
  - backend battle, settlement, and service integration plans
- `plans/frontend/`
  - frontend implementation plans
- `plans/gameplay/`
  - gameplay systems implementation plans
